Multi-Node IoT · MQTT Sensing

멀티노드
환경 센싱 모니터

여러 공간에 둔 센서 노드가 공기질(미세먼지·CO₂·VOC)을 측정해 MQTT로 발행하면, 허브가 한 데이터베이스에 모아 웹 대시보드로 실시간 표시하는 멀티노드 IoT 프로젝트.

UNO R4 WiFi Nano ESP32 SEN55 · SCD30 MQTT mosquitto · HiveMQ UNO Q · SQLite Streamlit 입문용 IoT 데모

01

프로젝트 요약

"여러 공간의 공기질을 한 화면에서 본다"를 측정 → 발행 → 수집 → 시각화로 직접 구성하는 프로젝트입니다.

각 공간에 둔 센서 노드(UNO R4 WiFi + SEN55 + SCD30)가 미세먼지(PM1.0/2.5/4/10)·CO₂·VOC·NOx·온습도 등 환경변수 11종을 측정해, NTP 정각정렬로 5분 평균 1건을 MQTT로 발행합니다. 브로커는 로컬(mosquitto) 또는 클라우드(HiveMQ) 중 현장 망 사정에 맞게 택일하고, 허브(Arduino UNO Q)의 hub.py가 메시지를 구독해 SQLite에 쌓으면 Streamlit 대시보드가 게이지·차트로 보여줍니다. 노드를 늘리면 대시보드에 칸이 자동으로 추가됩니다.
